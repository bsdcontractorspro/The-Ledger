# The Ledger — Money Calculators

Free calculators for the real financial decisions: mortgages, retirement, debt payoff, budgets, and AI costs. **No sign-up, no fluff — just the numbers you came for.**

**All calculations run client-side in your browser. No data is sent anywhere.**

## Features

- **Mortgage Payment Calculator** — See your monthly P&I, total interest, and lifetime cost
- **Retirement / Compound Growth** — Project future savings with monthly contributions and returns
- **Debt Payoff** — Calculate time-to-payoff and total interest on credit cards or loans
- **50/30/20 Budget** — Split your income into needs, wants, and savings
- **AI Token Cost** — Estimate LLM API costs before running production workloads

## Deploy on Netlify (30 seconds)

1. Download this repo or fork it on GitHub
2. Go to **netlify.com** → **Add new site** → **Deploy manually**
3. Drag & drop this folder, or connect your GitHub repo
4. Click **Deploy** — done. Your site is live.

## Local Development

Open `index.html` in any modern browser. No build step needed.

To serve locally:
```bash
python -m http.server 8000
# or
npx http-server
```

Then visit `http://localhost:8000`
